# 🚀 Machine Learning & Deep Learning Journey

Selamat datang di repositori ini! Repositori ini berisi kumpulan catatan, eksperimen *code*, dan proyek-proyek yang saya kerjakan selama mempelajari Machine Learning dan Deep Learning.

## 📂 Struktur Repositori

Repositori ini dibagi menjadi beberapa bagian berdasarkan tahapan materi:

*   **`01_Machine_Learning_Dasar/`**: Berisi implementasi algoritma dasar seperti Regresi, Klasifikasi (SVM, Random Forest), dan Scikit-Learn.
*   **`02_Deep_Learning_Fundamentals/`**: Berisi fundamental Neural Network, eksperimen bias-variance, dan pengenalan *framework* PyTorch.
*   **`03_Computer_Vision/`**: *(Akan datang)* Berisi proyek pemrosesan gambar dan *Convolutional Neural Networks (CNN)*.

## 🛠️ Teknologi yang Digunakan

*   **Bahasa:** Python 3
*   **Frameworks & Libraries:** PyTorch, Scikit-Learn, Pandas, NumPy, Matplotlib
*   **Environment:** Google Colab / Jupyter Notebook
